Detailed Description

Classe que simula o envio de dados da máquina real.

Esta classe tem como objetivo auxiliar o teste do socket, simulando o envio de dados da máquina real.

Constructor & Destructor Documentation

◆ machineClient()

machineClient::machineClient ( )

Construtor da classe machineClient.

Este construtor cria uma conexão socket com o servidor da interface, o qual é aberto pela classe receiveData. Lembrando que a classe ThreadController que cria objetos da classe receiveData.

◆ ~machineClient()

machineClient::~machineClient ( )

Destrutor da classe machineClient.

Este destrutor encerra a conexão com o servidor da interface, fechando o socket. Isto é feito na função close(this->socket_id);

Identifica se ocorreu um erro na conexão com o servidor da interface.

O valor 1 significa que um erro ocorreu e que deve ocorrer uma nova tentativa de reconexão. O valor 0 significa que não houve erro.
