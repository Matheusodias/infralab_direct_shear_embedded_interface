Detailed Description

Essa union tem como objetivo facilitar o envio de mensagem da máquina para a interface.

Utilizando a union, a transformação dos dados para string (unsigned char [] neste caso) é facilitada, pois os dados ocupam a mesma região de memória que a string. Assim para enviar os dados pelo socket basta enviar a variável payload.
